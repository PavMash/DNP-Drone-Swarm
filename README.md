# DNP Drone Swarm

## Модель

- Есть 2D поле с координатами (квадратная или прямоугльная форма)
- Есть произольное число дронов, которые спавнятся на этом поле (примерно 1-100 дронов, мб больше)
- У всех дронов одинаковый радиус распространения сигналов, в пределах которого они могут общаться с другими дронами
- Цель лидера роя - привести рой в центр поля и сформировать фигуру из дронов (фигура выбирается в зависимости от численности роя)
- Если в пределах радиуса коммуникации дрона нет других дронов, он считает себя лидером роя (из одного дрона)
- Сигналы от лидера роя и между дронами распространяются от соседей к соседям, покрывая в конечном итоге весь рой
- Лидер роя периодически посылает сигналы о том, что он жив. Лидер может умереть или дроны в рое могут счесть его мертвым, если давно не получали от него сигнал. Тогда запускается процесс нового лидера
- При объединении двух роев также запускается процесс выбора лидера объединённого роя

## Глобальные таски

- Написать агента и его логику обработки сообщений и обмена сообщений с соседями
- Написать среду, в которой работают агенты (поле, пулл агентов, инициализация)
- Имплементировать Gossip алгоритм для глобального обмена сообщениями в рое
- Имплементировать алгоритм выбора лидера
- Импламентировать алгоритм построения фигуры
- Визуализация через PyGame или HTML
- Репорт
